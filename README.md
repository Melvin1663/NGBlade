# **NGBlade** (By Melvin)
An api to get general historical data of ng players and guilds
## Endpoints
**API Url:** ng-blade.herokuapp.com
> ### **`GET`** `/player/:name`
> Gets a players historical data

> ### **`GET`** `/guild/:name`
> Gets a guilds historical data
## Response Codes
|Code|Meaning|
|--|--|
|200|Success|
|404|Player/Guild Not Found|
